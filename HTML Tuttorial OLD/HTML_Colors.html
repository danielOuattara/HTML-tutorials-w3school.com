HTML Hex Values Colors

Le haut de page

HTML color can be specified using a Hexadecimal value in the form: #rrggbb

rr := red value, gg:= green, bb:= blue value

Those Hexadecimal values ranges between 00 and ff

HTML HSL Values Colors

Le haut de page

H := hue, S := saturation, L := lightness

* Hue is a degree on the color whell from 0 to 360 || 0 := red, 120:= green, 240:= blue

* Saturation is a percentage value || 0% := shade of gray 100% := full color.

* Lightness is also a percentage || 0% := black 50% := neither light or dark100% := white
